Purpose

The purpose of this mod is to provide a contact page that visitors to your site can use to easily contact you.

Features

This contact page has several features built in.

  • It checks to see if all fields have been entered.  If not, it gives a warning to go back and fill in the necessary information.
  • It includes an anti spam code that a user must input correctly before the form will be sent.  This helps to prevent bots from automatically spamming your contact page.
  • The email address that the contact form is sent to is hidden in the source code.  This helps to prevent bots from learning your email address and spamming it.
  • It includes a character count that limits how much can be typed.  The default is 500 characters but can easily be changed - see below.
  • It includes an automatic spam filter that detects known words used in spam emails.  If any of the words are detected then the form will not be sent and the user will automatically be redirected to the website of your choice.

Upgrading

To upgrade from version 1.0 to version 1.1 simply upload the contact.asp file and allow it to overwrite your existing one.  Note that if you made any custom changes to the v1.0 contact.asp file you will need to reapply them to the v1.1 contact.asp file.

Optional Changes

The changes below are optional.  If you would like to apply them then go ahead.  Otherwise you are finished.

**************************************************

If you would like to change the number of maximum characters for the message input, then open contact.asp and go to line 253.  You will see this line:

" maxlength: 500," & vbNewLine & _

Change the number in red to the maximum number of characters you will allow.

**************************************************

If you would like to add words to your spam filter then open contact.asp and go to line 121.  You will see some words already listed.  Note that some may find these words offensive.  However, they are listed because these are words commonly used by spammers. Simply add your own words to the ones that are already there.  Be sure to put a comma after each new word.  DO NOT put spaces between them.

You can also change where the spammer is redirected if they use any of the listed words.  Simply change the url on line 136 of contact.asp.

Alternatively, if you do not want the anti spam filter, you can simply comment out or delete lines 120 - 138.  To comment out the lines, place a ' in front of each line from 120 - 138.
